# MDK 02.02: Программа расчета стоимости и формирования чека

## Назначение конфигурации
Программа предназначена для автоматизации расчета стоимости товаров, продаваемых на квадратный метр (например, отделочные материалы, ткани), с последующим формированием фискального чека.

## Установка и запуск
1.  Убедитесь, что на компьютере установлена платформа **1С:Предприятие 8.3**.
2.  Запустите конфигуратор 1С:Предприятие.
3.  Выберите пункт меню «Конфигурация» → «Создайте новую конфигурацию».
4.  Запустите и в вкладке "Администрация" и выберите загрузить информационную базу.
5.  Укажите файл с конфигурацией данной программы в формате (`.dt`).
6.  После загрузки обновите конфигурацию базы данных и запустите ее в режиме «1С:Предприятие».

## Использование
1.  Запустите информационную базу в режиме предприятия.
2.  Для начала работы необходимо предварительно заполнить справочник **«Материалы»** через соответствующий пункт меню.
3.  Основная работа ведется через документ **«Оплата»**, который создает новый чек.
4.  Следуйте пошаговой инструкции выше, чтобы провести полный цикл продажи.
